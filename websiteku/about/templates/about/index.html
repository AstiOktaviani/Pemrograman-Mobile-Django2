About Django

Django merupakan web framework open source berbasis python, digagas oleh Jacob Kaplan Moss dan Adrian Holovaty. Django menjadi salah satu web framework terbesar yang dimiliki python baik dari segi komunitas maupun web framewok itu sendiri . Django memiliki sebuah badan yang dinamakan dengan Django Software Fundation yang bertugas mengelolah pendanaan untuk kelagsungan pembangunan Django. Django memiliki tagline "The web framework for prefactionists with dedlines", bagaimana tidak karena secara default Django sudah memiliki berbagai modul umum yang bisa digunakan ketika mengembangkan aplikasi web. Dengan Django saja, anda sudah dapat menggunakan autentikasi sendiri, dan mendapatkan halaman admin sesuai dengan model yang kamu definisikan didalam aplikasi! Django banyak dipakai oleh perusahaan Internet besar belakagan ini seperti instagram, Discus, Mozzila dan masih banyak lagi. Django di untuk memudahkan developer membuat aplikasi web dengan cepat, mudah dan dengan kode yang lebih sedikit.

Django awalnya dikembangkan pada tahun 2003 dan 2005 oleh berbagai web developer yang bertugas membuat dan merawat web portal (newspaper website). Setelah membuat beberapa website, orang-orang tersebut mulai membuat ulang kode-kode yang pernah mereka tulis dengan menerapkan beberapa desaign pattern. Lalu disebarkan sebagi project open source dengan nama "Django" pada bulan juli 2005.
